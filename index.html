Projects

Twitter Sentiment Analyzer A Naïve Bayesian classifier that collects tweets related to a search query and categorizes each as positive or negative, plotting the results over a 7-day period.
Fantasy Basketball Drafter A fantasy basketball drafter that uses a heuristic algorithm based on teams' offensive & defensive ratings combined with gambling odds to predict individual player production. It then uses an integer linear programming solver to optimize a daily lineup for the popular fantasy sports website DraftKings.
Wikipedia Pathfinder An app that discovers paths between two Wikipedia articles. It uses a best-first search algorithm to prioritize more likely successful articles in its traversal to maximize time efficiency.
Spam Filter A spam-detection script that uses random forests to detect if mail is spam or not.
